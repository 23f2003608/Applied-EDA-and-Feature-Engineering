# EDA & Feature Engineering Portfolio

A comprehensive collection of **Exploratory Data Analysis** and **Feature Engineering** projects demonstrating data insights extraction, visualization, and preprocessing techniques for real-world datasets.

---

## 📊 Projects Overview

### 1. **Black Friday Dataset Analysis**
Retail sales analysis focused on customer purchase behavior during Black Friday events.

**Key Insights:**
- Customer demographics analysis (age, gender, city categories)
- Product category and purchase pattern exploration
- Sales contribution by customer segments
- Feature engineering: numerical encoding of categorical variables

**Techniques:** Missing value handling, data merging, categorical encoding, statistical analysis

---

### 2. **Flight Price Prediction EDA**
Time-series and regression-focused analysis of airline ticket pricing data.

**Key Insights:**
- Journey duration and route analysis
- Temporal patterns (date, month, year extraction)
- Airline and airport hub analysis
- Price variation across different flight attributes
- Feature engineering: temporal decomposition

**Techniques:** Time series feature extraction, multi-source data integration, outlier detection

---

### 3. **Zomato Restaurant Analytics**
Market analytics exploring restaurant ratings, cuisines, and delivery performance.

**Key Insights:**
- Missing value patterns and data quality assessment
- Restaurant ratings and review distributions
- Cuisine and location-based clustering
- Delivery time and cost analysis
- Feature correlation and relationship mapping

**Techniques:** Missing value analysis, heatmap visualization, categorical grouping, distribution analysis

---
